# Citation Display Policy

**Directive owner:** Papa  
**Original directive:** 2026-05-21  
**Reinforced:** 2026-06-10

NebulaMind wiki rendering uses inline evidence badges only.

Inline evidence badges only — this is the durable display rule.

## Rules

- Render citation markers such as `<!--cite:N-->` and `<!--cite:N,M-->` as inline evidence badges/popovers.
- Do not render numbered superscripts for citations.
- Do not render `[n]` reference markers in prose.
- Do not render a bottom `References`, `Bibliography`, or numbered citation list section on any wiki page.
- Keep canonical citation markers in stored content; this policy only governs display/rendering.

If a design doc or implementation plan conflicts with this policy, this policy wins.
